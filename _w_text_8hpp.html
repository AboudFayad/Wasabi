Detailed Description

Text rendering implementation.

Text rendering is Wasabi is done by using a 3rd party library called stb. stb is able to load a font (.ttf) file and create a bitmap of the characters of that font, with metadata about each character. With the information from stb, One can easily render text by rendering a quad for each letter, texturing it with part of the font bitmap (by specifying the uv coordinates for that letter).
